**- Файли README як ҳуҷҷат аст, ки маълумот дар бораи як лоиҳаро пешниҳод мекунад. Он аксар вақт аввалин файле аст, ки корбар вақте ба репозитория ташриф меорад мебинад, ки муҳим аст барои шенасоии лоиҳа ва роҳнамоӣ ба корбарон оиди истифода ё саҳм гузоштан дар он.**

## 2. Мақсадҳои асосии файли README

![key readme file](https://support.swiftify.com/hc/article_attachments/7586306507026)

**1. Шенасоии лоиҳа: Ташреҳ медиҳад, ки лоиҳа дар бораи чӣ аст ва мақсади он.**  
**2. Роҳнамои истифода: Маълумот дар бораи чӣ гуна созмон додан ва истифодаи лоиҳа.**  
**3. Роҳнамоӣ барои саҳмгузорӣ: Кӯмак мекунад, ки дигарон фаҳманд, ки чӣ тавр онҳо метавонанд дар лоиҳа саҳм гузоранд.**  
**4. Маълумоти тамос: Роҳҳои барои пайваст шудан ба нигоҳдорон барои дастгирӣ.**  
**5. Маълумоти иҷозатнома: Нишон медиҳад, ки лоиҳа чӣ гуна истифода ё тақсим карда мешавад.**

## 1. Чӣ аст Git?

![git](https://phoenixnap.com/kb/wp-content/uploads/2021/07/git-distributed-architecture.png)

**- Git як системаи назорати версияи тараҳҳудӣ мебошад, ки ба барномасозон имкон медиҳад, ки тағйиротҳоро дар файлҳо пайгирӣ кунанд, дар лоиҳаҳо ҳамкорӣ кунанд ва манбаи кодро самаранок идора кунанд. Он дар рушд ва нигоҳдории лоиҳаҳо барои нигоҳ доштани таърихи тағйирот ва мувофиқати коргарон истифода мешавад.**

## 2. Хусусиятҳои асосии Git

![git](https://d8it4huxumps7.cloudfront.net/uploads/images/6523cc0e62029_git_vs_github_01.jpg?d=2000x2000)

***Нақши Назорати Версия:***

**Ҳар як тағйироте, ки дар як лоиҳа рӯй медиҳад, пайгирӣ мекунад.  
Иҷозат медиҳад, ки версияҳои қаблиро баргардонем, агар лозим бошад.**

***Шоха ва Эҷоди Қатъ:***

**Барномасозон метавонанд дар шохаҳои алоҳида кор кунанд барои хусусан хусусиятҳо ё ислоҳи иштибоҳҳо.  
Тагйирот метавонанд пас аз баррасӣ ба шохаи асосӣ пайваст шаванд.**

***Системаи Тараҳҳудӣ:***

**Ҳар як барномасоз нусхаи пурраи репозиторияи лоиҳаро дорад.  
Кор кардан онлайн ё оффлайн ва ҳамкориро ба хубӣ пеш мебарад.**

***Ҳамкории Самаранок:***

**Барномасозон метавонанд бо ҳам дар як код кор кунанд бидуни дағалиятҳо.  
Git пайгирӣ мекунад, ки кӣ ва кай кадом тағйиротро овард.**

## 3. Терминологияи умумии Git

![git](https://www.dasca.org/content/images/main/git-terminology.jpg)

***Репозитория (Repo):***

**Як фолдери лоиҳа, ки файлҳо ва маълумоти пайгирии Gitро дар бар мегирад.**

***Коммит:***

**Як тасвири лоиҳаи шумо дар як лаҳзаи муайян.  
Командо: git commit -m "Тавсифи тағйирот"**

***Шоха:***

**Як версияи параллелии репозитория.  
Командо: git branch <branch-name> барои эҷоди шохаи нав.**

***Қатъ:***

**Тагйиротро аз як шоха ба шохаи дигар пайваст кардан.  
Командо: git merge <branch-name>**

***Пул:***

**Тагйиротро аз репозиторияи онлайн барои репозиторияи маҳаллӣ гирифтан ва пайваст кардан.  
Командо: git pull**

***Пуш:***

**Тагйиротро аз репозиторияи маҳаллӣ ба репозиторияи онлайн бор кардан.  
Командо: git push**

***Клон:***

**Нусхаи маҳаллӣ аз репозиторияи онлайн эҷод мекунад.  
Командо: git clone <repository-url>**

***Минтақаи Маҳкамшавӣ:***

**Як фазои муваққатӣ, ки тағйирот барои коммит омода мешавад.  
Командо: git add <filename>**

## 1. Чӣ аст GitHub?

![github](https://d2ds8yldqp7gxv.cloudfront.net/Blog+Explanatory+Images/How+to+use+github+1.webp)

**- GitHub як платформаи онлайн барои назорати версия ва ҳамкорӣ мебошад, ки бар асоси Git сохта шудааст. Он ба барномасозон имкон медиҳад, ки кодҳои худро ҷойгир кунанд, бо дигарон ҳамкорӣ кунанд ва лоиҳаҳоро самаранок идора кунанд. GitHub воситаҳоро барои кор кардан бо репозиторияҳои Git таъмин мекунад, ки ҳамкориро ва назорати версияро барои барномасозони ҷаҳонфурўш пешниҳод мекунад.**

## 2. Хусусиятҳои асосии GitHub

![github](https://images.javatpoint.com/tutorial/git/images/what-is-github.png)

***Ҷойгиркунии Репозитория:***

**Кодро нигоҳ медорад ва таърихи версияҳоро пайгирӣ мекунад.  
Метавонад репозиторияҳои ҷамъиятӣ ё хусусиро ҷойгир кунад.**

***Воситаҳои Ҳамкорӣ:***

**Тавсеаҳои сессияҳо, баррасии код ва масъалаҳо кӯмак мекунанд, ки гурӯҳҳо бо ҳам кор кунанд.**

***Менежменти Шоха:***

**Барномасозон метавонанд дар шохаҳои ҷудо кор кунанд ва тағйиротҳоро ба шохаи асосӣ пайваст кунанд.**

***Интегратсия ва Автоматизация:***

**Бо абзорҳои CI/CD, GitHub Actions ва APIҳои хориҷӣ ҳамкорӣ мекунад барои беҳтар кардани равандҳои рушд.**

***Ҷомеа ва Кодҳои Озод:***

**GitHub як маркази лоиҳаҳои кодҳои озод бо миллионҳо саҳмгузор аст.**

## 3. Терминологияи асосии GitHub

![github](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRxFJAwVREsnTUx-Bqco4e0XyARnpG7j0J2kg&s)

***Репозитория (Repo):***

**Як фолдери лоиҳа, ки код ва таърихи онро пайгирӣ мекунад.**

***Тавассути Мошини Ҷамъшавӣ (PR):***

**Як роҳ барои пешниҳод кардани тағйирот ба репозитория.  
Имкон медиҳад, ки баррасии код пеш аз пайваст кардан ба шоха.**

***Масъала:***

**Як вазифа, иштибоҳ ё дархости хусусият дар лоиҳа.**

***Форк:***

**Як нусхаи репозиторияи шахси дигар, ки имкон медиҳад шумо тағйиротро мустақилона анҷом диҳед.**

***Клон:***

**Нусхаи маҳаллӣ аз репозиторияи GitHub.**

***Ситора:***

**Як роҳ барои нишон кардани репозиторияҳои ҷолиб.**
